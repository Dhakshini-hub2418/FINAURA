# FinAura

FinAura is a desktop application that visualizes your financial "aura"—a reflection of your habits, moods, and decisions. It provides AI-driven insights and reflections to help you understand your relationship with money in a more holistic, emotional, and actionable way.

## Features
- Modern Electron + React UI
- Dashboard for financial activity and aura trends
- Financial Reflection component for personalized feedback
- Placeholder for AI-driven analysis and insights

## Getting Started

1. Install dependencies:
   ```
npm install
   ```
2. Start the app:
   ```
npm start
   ```

## Project Structure
- `public/electron.js`: Electron main process
- `src/App.js`: Main React app
- `src/components/`: UI components

## Customization
Replace placeholders with your own data sources and AI logic to personalize the experience.
